# 귤귤 일정관리 - main

1. Supabase SQL Editor에서 `supabase/gyulgyul_real_setup.sql` 전체 실행
2. 이 폴더를 GitHub 새 레포에 업로드
3. Vercel에서 해당 레포를 Import
4. 배포 후 `index.html`, `signup.html`, `dashboard.html` 경로 확인

로그인:
- 회원 로그인: 이름 + 전화번호
- 회원가입: `signup.html?token=토큰값`

주의:
- Supabase 프로젝트는 main/admin 둘 다 같은 프로젝트를 씁니다.
- SQL은 한 번만 실행하면 됩니다.
